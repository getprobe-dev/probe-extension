# PRobe Privacy Policy

The canonical, up-to-date privacy policy is published on the PRobe website:

**https://getprobe.dev/privacy-policy**

The version below is a summary for quick reference. If there is any discrepancy, the website version takes precedence.

---

## Summary

- All user data (API keys, chat history, pending comments, skill cache) is stored **locally on your device** via Chrome storage APIs.
- PRobe sends data only to **Anthropic** (AI responses), **GitHub** (PR data and comments), and **Google Fonts** (typography). A CORS proxy forwards Anthropic requests without logging or storing any data.
- PRobe has **no analytics, telemetry, or tracking**.
- You can clear all stored data at any time through the extension popup or by uninstalling the extension.
